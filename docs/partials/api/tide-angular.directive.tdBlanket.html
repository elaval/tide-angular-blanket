tdBlanket
directive in module tide-angular

Description

Generates charts to represent frequency of subjects with certain score (x attribute)

Consider the following example:

 <div td-blanket 
  td-data="controller.data" 
  td-category-attribute="'category'" 
  td-x-attribute="'x'" 
  td-size-attribute="'size'" 
  td-sort-criteria="controller.sortCriteria" 
  td-chart-type="controller.chartType" 
  td-x-label="'SIMCE'" 
  td-y-label="'Grupo de asistencia'">
 </div>

[inside "controller"]
this.data = [
 {"category":"A", "x": 200, "size": 5},
 {"category":"A", "x": 210, "size": 23},
 ...
 {"category":"B", "x": 200, "size": 1},
 {"category":"B", "x": 210, "size": 17},
 ...

];

this.sortCriteria = function(d) {return d};
this.chartType = "parallel curves";

Dependencies

  • tide-angular
  • tdBlanketLayout

Usage

as attribute
<div td-blanket
     td-data="{array}"
     td-category-attribute="{string}"
     td-x-attribute="{string}"
     td-size-attribute="{string}"
     td-sort-criteria="{function}"
     td-chart-type="{string}"
     td-x-label="{string}"
     td-y-label="{string}">
   ...
</div>

Parameters

ParamTypeDetails
tdDataarray

Data array used for populating the chart

tdCategoryAttributestring

Name of the attribute used for the X values in data objects

tdXAttributestring

Name of the attribute used for the X values in data objects

tdSizeAttributestring

Name of the attribute used for defining the size of the bubbles

tdSortCriteriafunction

Function used to define sort order of categories. Receives a category name and returns a string/number used for the rigth sort order

tdChartTypestring

Type of chart to be displayed ("bars", "parallel curves", "stacked curves")

tdXLabelstring

Label that describes variable used for x Axis (scores)

tdYLabelstring

Label that describes variable used for y Axis (categories)
